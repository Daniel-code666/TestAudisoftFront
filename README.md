# TestAudisoft Frontend

Aplicación web desarrollada en Angular para consumir la API de TestAudisoft y gestionar la información de estudiantes, profesores y notas.

La aplicación permite consultar, crear, editar y eliminar registros de cada entidad mediante una interfaz sencilla con navegación lateral, tablas con filtros y paginación, formularios en modales y confirmación de acciones importantes como la eliminación.

También incluye visualización de relaciones entre entidades, por ejemplo:

- consultar las notas asociadas a un estudiante
- consultar las notas asociadas a un profesor
- crear y editar notas seleccionando estudiantes y profesores desde campos buscables

Además, cuenta con manejo de carga mediante spinner, consumo centralizado de servicios HTTP y visualización de los mensajes de error enviados por el backend.

## Instalar dependencias luego de clonar el repo

npm install

## Ejecutar aplicación

ng serve -o
